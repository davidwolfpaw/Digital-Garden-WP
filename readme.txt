== Description ==

The Digital Garden Plugin allows you to create and manage a digital garden within your WordPress website. It provides a custom post type for creating notes and supports linking between notes with automatic backlink generation.

== Installation ==

1. Upload the `digital-garden-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

1. After activating the plugin, you'll find a new menu item called 'Notes' in your WordPress admin dashboard.
2. Click 'Notes' to create and manage your digital garden notes.
3. You can link between notes by using Markdown or any other method you prefer. Backlinks are automatically generated.

== Frequently Asked Questions ==

= How do I create and manage notes? =

After activating the plugin, go to the 'Notes' menu item in your WordPress admin dashboard. From there, you can create, edit, and manage your digital garden notes.

= How do I link between notes? =

You can create links between notes by using Markdown or any other method you prefer. When you link from one note to another, the plugin will automatically generate backlinks.
